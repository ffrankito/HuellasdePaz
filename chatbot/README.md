# Huellas de Paz — Chatbot

Asistente virtual con IA. **Fase 4 — no se desarrolla hasta que Fases 0-3 estén en producción.**

## Stack previsto

| Capa | Tool |
| --- | --- |
| Framework | Next.js 14 App Router + TypeScript |
| AI | Vercel AI SDK + Claude claude-sonnet-4-6 |
| Styling | Tailwind CSS 4 |
| Hosting | Vercel |

## Funcionalidad prevista

- Responde FAQs: planes, servicios, precios, ubicación
- Ayuda a elegir plan
- Captura datos básicos de contacto
- Escala a WhatsApp humano cuando no puede responder
- Tono empático para el contexto de duelo por mascotas
